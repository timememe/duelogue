# Промпты для генерации спрайтов и видео-анимаций по стейтам (Ace Attorney-стиль)

**Назначение:** готовые промпты image-gen / video-gen под каждый анимационный стейт из
[ace_attorney_animation_states.md](ace_attorney_animation_states.md), §2. Персонаж — референсный
(прикладывается отдельно как image reference), поэтому промпты **не описывают внешность/лицо/
одежду** — только эмоцию, атмосферу, свет, направление движения. Задача image-промпта — зафиксировать
кадр-настроение; задача video-промпта — оживить именно эмоциональный ритм стейта, а не конкретную
геометрию позы.

**Общий стилистический хвост** уже вшит в конец каждого промпта ниже (фон — чистый белый, для
удобного вырезания спрайта):
`plain solid white background, no scenery, visual novel courtroom drama style, bold clean linework,
dramatic rim lighting, anime key-art shading, half-body character portrait, high contrast mood
lighting`

## Заметки по использованию

- Референс-изображение персонажа задаёт внешность/дизайн — промпты выше **намеренно** не
  конфликтуют с ним описанием лица/одежды/точной геометрии позы.
- Фон везде — чистый белый (`plain solid white background, no scenery`), чтобы спрайт было легко
  вырезать/накладывать в движке без дополнительной обработки.
- Если генератор видео требует стартовый кадр — использовать соответствующий image-стейт как
  first frame, а video-промпт как motion-инструкцию поверх него.
- Talking-варианты стейтов рассчитаны на **луп** (idle loop поверх речи), Objecting/Desk Slam/
  Damage/Surprised — на **одноразовый импульсный клип** (see [ace_attorney_animation_states.md](ace_attorney_animation_states.md) §3).
