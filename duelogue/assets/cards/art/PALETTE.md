# DUELOGUE card-art palette

Основной язык: театральная печатная графика — линогравюра, сдержанное ар-деко,
тёмная бумага и тёплая костяная краска.

| Роль | Цвет | Hex |
|---|---|---|
| Чернильный фон | Ink | `#10131A` |
| Подложка окна иллюстрации | Art well | `#05080C` |
| Светлая линия и бумага | Bone | `#F2ECE3` |
| Тезис | Verdigris | `#43C59E` |
| Разбор | Cinnabar | `#E45B5B` |
| Кража | Brass | `#E5B84B` |
| Установка | Azure | `#57A3E3` |
| Именной приём | Amethyst | `#A875E8` |

## Рабочий формат v2

Все runtime-атласы — строго `2048×2048`, сетка `2×2`, ячейка `1024×1024`:

- `types_atlas_v2.png` — Тезис, Разбор / Кража, Установка.
- `thesis_atlas_a_v2.png` — Здравый смысл, Статистика / Определение, Аналогия.
- `thesis_atlas_b_v2.png` — Пример, Авторитет / Традиция, Эмоция.
- `razbor_atlas_a_v2.png` — Не в кассу, Передёрг / Контрпример, До абсурда.
- `razbor_atlas_b_v2.png` — Источник?, Корреляция / Ложная аналогия, А докажи.
- `theft_setup_atlas_a_v2.png` — Разворот, Та же логика / Рамка, Тезис дня.
- `setup_atlas_b_v2.png` — Позиция, Постулат / Принцип, Аксиома.
- `named_atlas_a_v2.png` — Gish gallop, Сократический вопрос / Ad hominem, Соломенное чучело.
- `named_atlas_b_v2.png` — Перенос бремени, Аксиома / две резервные пустые ячейки.

Исходные `v1` оставлены рядом как визуальные референсы. Runtime использует только `v2`.

При выдаче ячейки `card_art.gd` отступает на 15 px внутрь от математической границы.
Это исключает разделительную линию, внешний контур и соседнюю ячейку из фильтруемой
области. Подложка `ArtFrame` использует `Art well #05080C`, соответствующий фактическому
почти чёрному фону сгенерированных эмблем, поэтому letterbox не образует серых полос.
